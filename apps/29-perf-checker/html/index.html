ページ速度チェッカー
Chrome拡張 - Performance APIによるページ計測ツール

閲覧中のページのパフォーマンスを即座に計測するChrome拡張です。 ブラウザの Performance API を使って、読み込み時間やリソースの状況を可視化し、改善ポイントを提示します。

計測項目

DOMContentLoaded
HTMLの解析が完了しDOMツリーが構築されるまでの時間
Load Time
画像やスタイルシート含む全リソースの読み込み完了時間
TTFB
Time to First Byte。サーバーからの最初のレスポンス到達時間
DOM要素数
ページ内の全DOM要素の数。多すぎるとレンダリングに影響
リソース集計
読み込まれたリソースの件数と合計サイズ
画像分析
画像ファイルの枚数と合計転送サイズ

スコア判定

各指標を総合的に評価し、100点満点のスコアと Good / Medium / Bad の3段階で判定を表示します。 基準値を超えた項目には具体的な改善提案を表示します。

インストール手順

1
chrome-ext フォルダを任意の場所に配置します。
2
Chrome で chrome://extensions を開きます。
3
右上の「デベロッパーモード」をオンにします。
4
「パッケージ化されていない拡張機能を読み込む」をクリックし、chrome-ext フォルダを選択します。
5
ツールバーに追加された拡張アイコンをクリックすると、現在のページの計測結果が表示されます。

使用技術

Performance API
ブラウザ標準のナビゲーション・リソースタイミング情報
Chrome Extensions API
Manifest V3、chrome.scripting.executeScript
activeTab権限
拡張クリック時のみアクティブタブにアクセス。常時監視はしない
