PDF比較ツール テスト用サンプルデータ

このフォルダには、ソフトウェア利用規約の2つのバージョンが含まれています。
PDF比較ツールの動作確認にお使いください。

ファイル一覧:
  v1-terms.txt  第1版 利用規約（2025年4月制定）
  v2-terms.txt  第2版 利用規約（2026年1月改定）

v2では以下のような変更が加えられています:
  - サービス概要にAIアシスタント機能の記載を追加
  - 利用条件に登録情報変更の届出義務を追加
  - 禁止事項を3項目追加（API濫用、競合利用、AI出力の無断転記）
  - 個人情報条項を大幅に強化（データ保管場所、漏洩時通知等）
  - 料金を改定（スターター 980円→1,280円、ビジネス 1,980円→2,480円）
  - 支払方法にPayPay請求書払いを追加
  - 解約予告期間を14日前→30日前に変更
  - 年間契約の中途解約違約金を新設
  - データ保存期間を30日→90日に延長
  - 規約変更時の事前通知義務を追加

テスト手順:
  1. v1-terms.txt と v2-terms.txt をそれぞれテキストエディタで開く
  2. 各ファイルの内容をPDFとして保存する
     - macOS: テキストエディット → ファイル → PDFとして書き出す
     - Windows: メモ帳 → 印刷 → Microsoft Print to PDF
  3. PDF比較ツールを開き、v1のPDFとv2のPDFをアップロードする
  4. 差分がハイライト表示されることを確認する
