画像一括変換ツール テスト用サンプルデータ

このフォルダには、画像一括変換の動作確認用SVGファイルが5つ含まれています。

ファイル一覧:
  sample-chart.svg    棒グラフ（四半期売上）  800x600px
  sample-logo.svg     企業ロゴ                800x600px
  sample-diagram.svg  承認フロー図            800x600px
  sample-icon.svg     アプリアイコン          512x512px
  sample-card.svg     名刺デザイン            910x550px

テスト手順:
  1. 画像一括変換ツールを開く
  2. このフォルダ内のSVGファイルを複数選択してアップロードする
  3. 変換先フォーマット（PNG, JPEG, WebP等）を選択する
  4. リサイズやクオリティの設定を変更してみる
  5. 一括変換を実行し、結果をダウンロードする

SVGファイルはブラウザで直接開いて内容を確認できます。
