# 画像一括変換ツール

## 概要
複数の画像ファイルをドラッグ&ドロップで読み込み、リサイズ・フォーマット変換・圧縮をブラウザ内で一括処理するツール。サーバーへのアップロードは不要で、すべてクライアントサイドで完結する。

## 機能
- 複数画像のドラッグ&ドロップ入力
- 幅・高さ指定によるリサイズ（アスペクト比維持オプション付き）
- フォーマット変換（PNG / JPEG / WebP）
- 圧縮品質スライダー（0.1〜1.0）
- 変換後プレビューサムネイル表示
- 個別ダウンロード / ZIP一括ダウンロード（JSZip使用）

## 技術的なポイント
- Canvas API で画像を描画し、toBlob() でフォーマット変換・圧縮を実現
- FileReader で画像をData URLとして読み込み
- JSZip でクライアントサイドZIP生成
- 3バージョン（HTML単体 / React+Vite / Next.js）で実装

## デザイン
- 白背景、ネイビー (#2c52a4) アクセント
- 日本語UI
- クリーンでミニマルな設計
