[稟議 1] 社内チャットツール導入
件名: 社内コミュニケーションツール(Slack Business+)の導入
起案部署: 情報システム部
起案者: 山本 拓也
決裁区分: B決裁(部長決裁)
概算金額: 720,000円/年 (60名 x 1,000円/月 x 12ヶ月)
契約期間: 2026年7月1日〜2027年6月30日(年間契約・自動更新)

理由:
- 社内メールの平均返信時間が4.2時間。チーム間の情報伝達に遅延が生じている
- プロジェクト横断のナレッジがメールスレッドに埋没し、検索・再利用ができていない
- リモートワーク比率が週3日に増加し、非同期コミュニケーション基盤が必要になった
- 競合ツール(Microsoft Teams)も検討したが、外部サービス連携の柔軟性と開発チームの既存利用実績からSlackを選定

期待効果:
- 社内コミュニケーションの応答時間を平均4.2時間から1時間以内に短縮
- チャンネルベースの情報整理によりナレッジの蓄積と検索性を向上
- 既存のメーリングリスト運用費(年間180,000円)を廃止し、実質的な追加コストは540,000円/年
- Slack Connect機能を活用し、主要取引先5社とのやりとりも集約可能

代替案:
- Microsoft Teams(月額750円/人): 既にOffice 365を契約中のため追加費用は低いが、外部連携が限定的
- Google Chat(月額680円/人): Google Workspace契約に含まれるが、スレッド管理機能が弱い
- 現状維持: コスト増はないが業務効率の改善は見込めない

[稟議 2] 開発チーム用ワークステーション購入
件名: ソフトウェア開発チーム向けワークステーション5台の購入
起案部署: 開発部
起案者: 鈴木 健一
決裁区分: A決裁(役員決裁)
概算金額: 3,250,000円(税込) (650,000円/台 x 5台)
納品希望日: 2026年5月末

理由:
- 現行開発機(2022年導入)のスペック不足により、ローカルビルド時間が平均12分に増加
- Docker環境でのマイクロサービス同時起動にメモリ不足(現行16GB)が頻発し、1日あたり平均3回の再起動が発生
- AI支援コーディングツール(GitHub Copilot等)のローカルモデル実行にGPUメモリが必要
- 中途採用2名の入社(2026年6月)に伴う端末確保も兼ねる

購入予定機種:
- Apple MacBook Pro 16インチ M4 Pro (48GB RAM / 1TB SSD)
- 単価: 598,000円(税抜) → 657,800円(税込)
- 購入先: Apple Japan法人販売(見積番号 AJ-2026-03-0892)
- 減価償却: 4年定額法(1台あたり年間162,500円)

期待効果:
- ローカルビルド時間を12分から3分に短縮(年間で開発者1人あたり約200時間の作業時間節約)
- Docker環境の安定化によるエンジニアの集中力維持とストレス低減
- 新規採用者への即座の環境提供(入社初日から開発着手可能)

代替案:
- Windows機(Dell Precision 5690): 1台あたり約480,000円で低コストだが、開発チームの技術スタックがmacOS前提
- クラウド開発環境(GitHub Codespaces): 月額約40,000円/人で初期投資は抑えられるがネットワーク遅延が課題

[稟議 3] 外部研修プログラム受講
件名: クラウドアーキテクチャ研修(AWS Solutions Architect対策含む)の受講
起案部署: 情報システム部
起案者: 渡辺 あゆみ
決裁区分: C決裁(課長決裁)
概算金額: 495,000円(税込) (165,000円/人 x 3名)
実施時期: 2026年5月〜7月(全6回、各回4時間、隔週土曜日)

理由:
- 現在オンプレミスで稼働している基幹システムのAWS移行プロジェクトが2026年度下期に予定
- 社内にAWS認定資格保有者がおらず、アーキテクチャ設計の判断をベンダーに依存している
- ベンダー選定・評価時に自社側で技術的な妥当性を判断できる人材が必要
- 対象者: 田中(インフラ担当)、中村(開発リーダー)、渡辺(PM) の3名

研修内容:
- 提供元: クラスメソッド株式会社「AWS実践アーキテクチャ設計コース」
- カリキュラム: VPC設計、EC2/ECS選定、RDS/Aurora運用、セキュリティ設計、コスト最適化、模擬試験
- 修了後、AWS Solutions Architect Associate試験を受験(受験料は別途11,000円/人)

期待効果:
- AWS移行プロジェクトにおける自社判断力の確保(ベンダー提案の妥当性を評価可能に)
- 3名がAWS SAA資格を取得すれば、社内の技術的信頼性が向上
- 研修で得たナレッジを部内勉強会で展開し、チーム全体のクラウドリテラシー底上げ
- 中長期的にはベンダー依存度の低減による運用コスト削減(年間推定600,000円)

代替案:
- Udemy等のオンライン講座(1人あたり約5,000円): コストは低いが体系的な学習とハンズオン環境の提供がない
- AWSトレーニング公式コース(1人あたり約300,000円): 質は高いが予算超過
- 独学+書籍: コスト最小だが学習効率が低く、プロジェクト開始に間に合わないリスク
