株式会社ネクストブリッジ・テクノロジーズ 採用情報

バックエンドエンジニア（シニア）

■ 募集背景
当社が運営するBtoB SaaSプラットフォーム「BridgeConnect」は、導入企業数が前年比180%で拡大中です。サービスの安定性とスケーラビリティを支えるバックエンド基盤の強化が急務となっており、設計から実装、運用まで一貫して担えるエンジニアを募集します。

■ 業務内容
- REST API / gRPC を用いたマイクロサービスの設計・開発
- PostgreSQL / Redis を活用したデータ層の設計・チューニング
- CI/CDパイプラインの構築と改善（GitHub Actions）
- 可観測性基盤（OpenTelemetry, Datadog）の整備
- コードレビューを通じたチーム全体のコード品質向上
- プロダクトマネージャー、フロントエンドチームとの技術要件の策定

■ 必須スキル
- Go または Python によるWebアプリケーション開発経験 3年以上
- REST API の設計・実装経験
- RDBMSを用いたデータベース設計経験（正規化、インデックス戦略）
- AWS上での本番サービス運用経験（EC2, ECS, RDS, S3 等）
- Git を用いたチーム開発経験

■ 歓迎スキル
- gRPC / Protocol Buffers の利用経験
- Kubernetes（EKS）上でのコンテナ運用経験
- テスト駆動開発（TDD）の実践経験
- パフォーマンスチューニングの経験（SQL最適化、キャッシュ戦略）
- テックリードまたはアーキテクトとしてのチームリード経験
- OSS へのコントリビューション実績
